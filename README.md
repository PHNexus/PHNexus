## 🧭 Journey

I started programming young, experimenting with HTML, CSS, and JavaScript to build small projects and study repos. Like many learners, I faced periods of inconsistency and stepping away from code. I returned with a clearer focus on backend development and Java, aiming to understand software engineering deeply rather than just surface-level coding.

Currently, I maintain a local AI environment using LM Studio, OpenRouter, and OpenClaude to experiment with offline and cloud‑assisted workflows. I’ve also set up a secondary PC (PC2) running Ubuntu LTS as a home server, where I practice SSH, networking, and basic backend infrastructure. My days are spent in terminal‑centric workflows, writing automation scripts, and strengthening core programming concepts through hands‑on projects.
