## 🧭 Journey

I started programming at a young age, experimenting with HTML, CSS and JavaScript through small projects and study repositories. Like many developers, I went through periods of inconsistency, stopping and restarting multiple times.

Instead of giving up, I came back with a clearer goal: focus on backend and truly understand software engineering — not just write code, but understand systems.

I began rebuilding my foundation with Java and backend concepts, while also exploring more advanced environments. I created a local AI setup using LM Studio, OpenRouter and OpenClaude, combining offline and cloud-based workflows.

At the same time, I experimented with infrastructure using a second PC (PC2) running Ubuntu LTS as a home server, working with SSH, networking and basic backend environments.

Today, I focus on strengthening fundamentals, writing better code, and building real systems step by step.
